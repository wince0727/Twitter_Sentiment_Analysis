# Twitter Sentiment Analysis

# Overview
This project performs **sentiment analysis on Twitter data** using Python. Tweets are analyzed using Natural Language Processing (NLP) techniques and classified into **Positive, Neutral, or Negative** sentiments.

# Objectives
- Load and preprocess Twitter text data  
- Apply sentiment analysis using TextBlob  
- Classify tweets into sentiment categories  
- Generate sentiment counts and export results  

# Tools & Technologies
- Python  
- Pandas  
- TextBlob  
- VS Code  

# Dataset
- `Tweets.csv` – contains tweet text data  
- `Sentiment_Output.csv` – generated sentiment results  

# Methodology
1. Loaded tweet data using Pandas  
2. Cleaned and processed text content  
3. Applied TextBlob to calculate sentiment polarity  
4. Labeled tweets as Positive, Neutral, or Negative  
5. Displayed sentiment distribution  

# Output

Positive : 12618  
Neutral  : 9428  
Negative : 5435
